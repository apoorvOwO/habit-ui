# ⚡ Habit Tracker Suite

A local-first, zero-telemetry productivity suite built for speed and focus. Track habits, manage complex builds, and stash notes. Everything lives in your browser's `localStorage`. No accounts, no cloud, no bloat.

Built with a heavy glassmorphism aesthetic and a custom techwear-inspired "Claude" dark mode. 

## 🛠️ Core Features

### 1. 📅 Habit Tracker
Visual, GitHub-style contribution grid to track daily progress.
- **Milestones & Streaks:** Track continuous progress (e.g., VO2 max improvements, hitting that 5km run under 25 mins).
- **Analytics:** Monthly consistency charts (Chart.js) and progress rings.
- **Log & Notes:** Daily timestamped journaling for each habit.
- **Data Portability:** 1-click CSV export.

### 2. 📋 Project Planner (Kanban)
Drag-and-drop board for multi-step projects.
- **Multiple Workspaces:** Tab between different projects instantly.
- **Card Metadata:** Add priority tags, deadlines, and categories. 
- *Use Case:* Perfect for staging hardware mods (e.g.,  software dev. goals etc)

### 3. 📝 Notes & Keep
Quick-capture brain dump space.
- **Rich Previews:** Auto-fetches OpenGraph data for links.
- **YouTube Integration:** Paste a link -> get an embedded playable video.
- **Clean Grid:** Masonry-style layout for high-density information.

## 🎨 Themes
Cycle through themes with the ☀️/🌙/🤖 toggle:
- **Dark Mode:** Deep blue/slate glassmorphism (Default).
- **Light Mode:** Clean, high-contrast white/gray.
- **Clause Theme:** High-contrast cyberpunk/techwear aesthetic (pure blacks, stark UI).
- **Dynamic Accents:** 🎲 button randomizes the global accent color using HSL generation.
